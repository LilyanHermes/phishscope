# PhishScope 🔍

Real-time phishing detection Chrome extension with explainable risk analysis.

## ✨ Features
- Real-time webpage analysis
- Explainable risk scoring (0–100)
- Threat level detection (Low / Medium / High)
- Detection of phishing indicators (domain spoofing, brand impersonation)
- Clean, modern UI with cybersecurity-focused design

## 🚀 How It Works
PhishScope analyzes the current webpage for common phishing indicators and assigns a risk score based on detected patterns.
